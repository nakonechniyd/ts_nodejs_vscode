# Web parsing script (typescript + node.js)

## Requirements
Look at [package.json](./package.json) `engines` section

## Install
```
npm ci
```

## Run tests once
```
npm test
```

## Run tests in watch mode
```
npm run watch:test
```

## Compile typescript to javascript
```
npm run build
```

## Run parser
```
npm start
```

## Debugging
```
press F5 in vscode
```

## Check types by typescript
```
npm run check:types
```

## Check linting by eslint
```
npm run check:lint
```
